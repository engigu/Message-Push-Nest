# 更新日志

## 2025

### 2025.12.17
- **新增飞书机器人渠道**
  - 支持飞书 Webhook 消息发送
  - 支持文本和 Markdown 格式
  - 支持加签验证和关键字过滤
  - 支持 @提醒（手机号、用户ID、全员）
- **阿里云短信支持动态接收者**
  - 支持群发模式，可在 API 调用时动态指定手机号列表
  - 支持固定接收者和动态接收者两种模式
  - 实例列表正确显示接收者状态
- **代码架构优化**
  - 将消息类型常量统一迁移到 `pkg/constant` 包
  - 使用注册模式重构渠道验证和测试逻辑
  - 优化前端接收者显示逻辑，基于配置动态判断
  - 修复循环导入问题，提升代码可维护性
- **前端优化**
  - 优化新增/编辑渠道弹窗高度，减少滚动
  - 优化渠道类型显示样式，更加简洁美观
  - 修复 Drawer 组件最大高度限制问题

### 2025.12.06
- **新增消息模板功能**
  - 支持创建可复用的消息模板
  - 支持占位符动态替换（`{{key}}` 语法）
  - 支持 Text、HTML、Markdown 三种格式
  - 支持为模板配置多个发送实例
  - 支持 @提醒功能（钉钉、企业微信）
- **新增 V2 API**
  - 基于模板的消息发送接口
  - 使用加密 Token 提升安全性
  - 支持占位符参数传递
  - 自动遍历模板的所有启用实例
